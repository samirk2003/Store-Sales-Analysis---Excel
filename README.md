# Vrinda Store Sales Analysis Dashboard

## Project Overview
This project analyzes sales data from an e-commerce store called Vrinda Store using Microsoft Excel.  
The goal of this project is to understand customer behavior, sales performance, and business trends using data visualization.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Slicers (Interactive Filters)

## Dashboard Features

The dashboard includes the following analysis:

1. Orders vs Sales Trend (Monthly analysis of orders and revenue)
2. Sales by Gender (Men vs Women purchasing behavior)
3. Order Status Distribution (Delivered, Cancelled, Returned, Refunded)
4. Top 5 States by Sales Contribution
5. Orders by Age Group vs Gender
6. Orders by Sales Channel (Amazon, Flipkart, Myntra, etc.)

## Key Insights

- Women customers contribute the highest share of orders.
- Adult customers place the majority of purchases.
- Maharashtra and Karnataka are the top performing states.
- Around 92% of orders are successfully delivered.
- Amazon and Flipkart contribute the highest number of orders.
